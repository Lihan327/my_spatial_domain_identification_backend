使用说明 · Quick Start ▾
  • 本页面通过相对路径引用 DLPFC/DLPFC_result/<slice_id>/ 下的 GT / Pred 图像，请通过本地静态服务器打开（避免浏览器 CORS 限制）。
  • 推荐启动方式：cd C:\MyCode\AI_training_1 && python -m http.server 8000，浏览器访问 http://localhost:8000/spatial_domain_frontend.html。

使用文档

完整使用说明

启动方式

cd C:\MyCode\AI_training_1
python -m http.server 8000
# 浏览器打开 http://localhost:8000/spatial_domain_frontend.html

交互快捷键

  • 按住亚克力板拖拽 — 自由旋转（5px 阈值避免误触）
  • R — 亚克力板旋转 90°
  • F — 翻面 (GT ↔ Pred)
  • O — 切换叠加模式 (显示错分点)
  • Esc — 重置视图
  • 1-9, 0, -, = — 切换 12 张切片

* 滚轮缩放、Overlay 模式下拖拽已禁用（防止与叠加红点冲突）。

Overlay 叠加模式

  • 点击 Overlay 按钮或按 O 开启叠加：板强制归零到正面，GT 图变 50% 透明，Pred 图以 50% 透明度叠加在上层 (multiply 混合)
  • 红点为真实像素差异检测结果（基于灰度差异），如果检测失败则不显示
  • 左/右的 Front / Back 标签会高亮当前正在查看的面，板下方也有"正在查看"提示
  • 开启 Overlay 后再按 R/F/Esc 或切换切片会自动关闭 Overlay

指标说明

  • ARI (Adjusted Rand Index) — 调整兰德指数，越接近 1 越好
  • NMI (Normalized Mutual Information) — 标准化互信息
  • HS (Homogeneity Score) — 同质性
  • CS (Completeness Score) — 完整性
